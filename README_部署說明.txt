MSForex Academy 網站部署包

已設定正式網域：
https://msforexacademy.com

本包包含：
- index.html
- registration_qr.png
- ken_photo.jpg
- robots.txt
- sitemap.xml
- CNAME

GitHub Pages：
1. 新建 repository
2. 上傳全部檔案到根目錄
3. Settings → Pages → Deploy from a branch
4. Branch 選 main，資料夾選 /(root)
5. DNS 設定：
   - 在網域註冊商新增 GitHub Pages 所需 A / CNAME 紀錄
6. 上線後到 Google Search Console 提交：
   https://msforexacademy.com/sitemap.xml

Cloudflare Pages / Netlify 也可直接上傳這包。
